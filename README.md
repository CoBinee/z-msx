## 説明
安全にパイルダーオンしてください。<br>
こちらから遊べます。⇒  [WebMSX](http://webmsx.org/?MACHINE=MSX1J&PRESETS=MSXMUSIC&ROM=https://github.com/CoBinee/z-msx/raw/main/rom/Z.ROM)

- `←`, `→`<br>左右に移動
- `SPACE`<br>上昇
- ジョイスティックにも対応

## 動作環境
- 16KB 以上の RAM を搭載した MSX
